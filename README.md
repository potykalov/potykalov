## 🚀 Проекты

### 🪙 [Неткоин — сайт-биржа для крипто-стартапа](https://github.com/potykalov/netcoin-exchange)
[▶️ Открыть демо](https://netcoin-exchange.onrender.com)
> Демо размещено на Render. Первый запуск может занять около минуты, если сервер был в спящем режиме.

**Год:** 2026  
**Статус:** завершён  
**Тип:** учебный дипломный проект

Сайт-биржа Неткоин - веб-приложение для управления внутренней валютой крипто-стартапа. Пользователь может зарегистрироваться, войти в личный кабинет, посмотреть баланс, пополнить счёт, конвертировать валюту, перевести средства другому пользователю и управлять списком избранных получателей.

**Что реализовано:**
- авторизация и регистрация пользователя;
- выход из личного кабинета;
- получение и отображение данных профиля;
- просмотр и обновление баланса;
- пополнение счёта;
- конвертация валют;
- перевод средств другому пользователю;
- получение и обновление курсов валют;
- добавление и удаление пользователей из избранного;
- вывод сообщений об успешных действиях и ошибках;
- синхронизация списка избранных пользователей со списком получателей для перевода.

**Стек:** `JavaScript` `API` `Node.js` `Git`

**Вывод:**  
Закрепил работу с клиентской логикой, обработкой пользовательских действий, взаимодействием с серверным API, обновлением интерфейса и обработкой успешных и ошибочных ответов сервера.
